# OBEscape — Claude Instructions

## Project Context

OBEscape is a mobile-first browser game. All features must work on mobile browsers without relying on keyboard input. Touch/pointer controls are the primary input method; keyboard support is a secondary convenience for desktop users.

## Milestones

`TODO.md` is the authoritative milestone and known-issues tracker. Check it before starting new feature work to understand what has been completed and what comes next.

## Git

- Always work on the `main` branch unless explicitly told otherwise.
- This branch is deployed to GitHub Pages.

## Plan Mode

- When plan mode is active, take **no** write actions of any kind: no file edits, no commits, no pushes, no shell commands that modify state.
- Read-only tools (Read, Grep, Glob, Bash reads) are permitted.
- Do not exit plan mode to make changes — wait for the user to approve the plan and exit plan mode explicitly.
